**#📊 Global Superstore Sales Analysis (Power BI)**

**📌 Project Overview**

- This project presents an interactive Sales Analysis Dashboard built using Power BI. It analyzes global superstore data to uncover key business insights such as sales trends, profit distribution, and customer behavior.

**🎯 Objectives**

- Analyze overall sales performance

- Identify top-performing products and regions

- Track profit trends and losses

- Understand customer purchasing patterns

- Provide actionable business insights

**📁 Dataset Details**

 | Field     | Description                                                                                                                                       |

 | --------- | ---------------------------------------------------------------------------------------------------------------------------  --------------------- |

 | File Name | global superstore sales.pbix                                                                                                                      |

 | Data Type | Sales Transactions                                                                                                                                |

 | Columns   | Order ID, Order Date, Ship Date, Customer Name, Segment, Country/Region, Product Name, Category & Sub-Category, Sales,    
 Quantity, Discount, Profit |

**📊 Key Features**

**Feature	Description**

- 📈 Sales Dashboard	Visual representation of total sales and trends

- 🌍 Regional Analysis	Sales & profit across different regions

- 🛒 Product Insights	Top-selling products and categories

- 💰 Profit Analysis	Profit vs Loss identification

- 📅 Time Analysis	Monthly & yearly sales trends

- 👥 Customer Insights	Customer segmentation and behavior

**🛠️ Tools & Technologies**

- Power BI

- Data Visualization

- Data Cleaning & Transformation

- DAX (Data Analysis Expressions)
